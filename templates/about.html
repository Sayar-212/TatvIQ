About Our HR-Tech Solution

Transforming HR processes with AI to improve hiring decisions and employee engagement.

Our Mission

The HR-Tech Innovation Suite was developed to address two critical challenges faced by modern HR departments:

  • Efficient Candidate Screening: Automating the process of matching candidate resumes to job requirements
  • Employee Retention: Analyzing employee feedback to identify sentiment trends and prevent attrition

By leveraging the power of artificial intelligence through Google's Gemini API, we've created tools that help HR professionals make data-driven decisions about hiring and employee engagement.

Key Features

Resume Screening

  • PDF & DOCX resume parsing
  • Skill extraction and matching
  • Match scoring against job requirements
  • Candidate strengths and weaknesses analysis

Sentiment Analysis

  • Employee feedback analysis
  • Sentiment scoring and theme extraction
  • Attrition risk assessment
  • Actionable engagement recommendations

Technology Stack

AI Technology

Powered by Google's Gemini API, our system leverages advanced language models for natural language understanding and processing.

Backend

Built with Flask, our backend handles document processing, API communication, and data analysis securely and efficiently.

Frontend

A responsive, intuitive interface built with Bootstrap and modern JavaScript provides a seamless user experience.

How It Works

Resume Screening Workflow

1
Upload Resume & Job Description

User uploads a resume file (PDF/DOCX) and enters the job description for a Software Engineer role.

2
Text Extraction

System extracts text content from the resume document using specialized parsers.

3
AI Analysis

Google Gemini API analyzes the resume text against the job description to identify matching skills and qualifications.

4
Results Generation

System compiles match scores, skills analysis, and overall assessment of the candidate's fit for the role.

5
Visual Presentation

Results are presented in an intuitive interface with visualizations and actionable insights.

Sentiment Analysis Workflow

1
Input Employee Feedback

User enters employee feedback text from surveys, reviews, or exit interviews.

2
Sentiment Processing

System sends the feedback text to the Google Gemini API for processing.

3
AI Analysis

Gemini API analyzes sentiment, identifies key themes, and assesses attrition risk factors.

4
Recommendation Generation

System generates tailored recommendations to address concerns and improve engagement.

5
Visual Presentation

Results are presented with charts, sentiment indicators, and actionable insights.
